# 🛍️ Vitrine de Ofertas - React + TypeScript

## 📌 Sobre o projeto

Este projeto foi desenvolvido como parte de um desafio técnico para vaga de Desenvolvedor Front-end Júnior.

A aplicação simula uma vitrine de produtos com layout inspirado em e-commerces reais, exibindo ofertas da semana de forma organizada e responsiva.

---

## 🚀 Tecnologias utilizadas

* React
* TypeScript
* Vite
* SCSS Modules
* React Router DOM

---

## 🎯 Funcionalidades

* Página `/ofertas` com listagem de produtos
* Exibição de:

  * imagem do produto
  * nome
  * preço
  * botão de compra
* Banner no topo
* Layout responsivo
* Componentização (ProductCard, ProductList, etc.)
* Navegação entre rotas

---

## 📁 Estrutura do projeto

```
src/
├── assets/          # Imagens e recursos estáticos
├── components/      # Componentes reutilizáveis
│   ├── ProductCard/
│   └── ProductList/
├── pages/           # Páginas da aplicação
│   └── Ofertas/
├── services/        # Simulação de API / dados mockados
├── types/           # Tipagens TypeScript
├── App.tsx          # Rotas da aplicação
└── main.tsx         # Ponto de entrada
```

## 🔧 Como rodar o projeto localmente

### 1. Clone o repositório

```bash
git clone https://github.com/SEU-USUARIO/desafio-tecnico-b8one.git
```

### 2. Acesse a pasta do projeto

```bash
cd desafio-tecnico-b8one
```

### 3. Instale as dependências

```bash
npm install
```

### 4. Rode o projeto

```bash
npm run dev
```

O projeto estará disponível em:

```
http://localhost:5173
```

## 🏗️ Build para produção

## 🧠 Decisões técnicas

* **TypeScript** foi utilizado para garantir maior segurança e previsibilidade no código
* **SCSS Modules** para evitar conflitos de estilo e manter organização
* **Componentização** para reutilização e separação de responsabilidades
* **HashRouter** foi utilizado para evitar problemas de roteamento em ambiente de deploy estático
